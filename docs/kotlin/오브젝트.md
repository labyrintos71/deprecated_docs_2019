# Object

코틀린에는 static 에 개념이 없습니다.
대신 object가 있습니디.
companion 오브젝트는 클래스에서 오브젝트로 선언한 부분만 싱글톤화 됨.

```kotlin
class tester {
        init {
            println("init")
            println("testfun" + this.hashCode())
        }

        fun testfun() {
            println("testfun" + this.hashCode())

        }

        companion object Foo {
            fun callFoo() = println("Foo " + this.hashCode())
            var foo = "foo"
        }
    }

    object Foo {
        fun callFoo() = println("Foo " + this.hashCode())
        var foo = "foo"
    }
```
